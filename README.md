### Run Locally

To run **Etarnet** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/ugoMusk/Etarnet.git
```

Windows:

```bash
git clone https://github.com/ugoMusk/Etarnet.git
```
